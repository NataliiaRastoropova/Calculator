# Calculation app
Calculate operations for equation (+,-,*,/). 

#### Request example for equation (1+2*3)
Operation length has to be less than numbers less for 1 character. 
```json 
{
  "Numbers": [1,2,3],
  "Operations": ["+","*"]
}
```

### Technologies
ASP.NET Core 3.1, Rest API, MongoDb, Swagger UI.
